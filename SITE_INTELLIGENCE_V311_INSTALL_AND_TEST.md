# Site Intelligence v3.1.1 — Install and Test

1. Place the v3.1.1 repository ZIP and installer in the same folder.
2. Run the macOS installer.
3. Deploy the latest `main` commit on Render.
4. Upload the generated v3.1.1 WordPress plugin ZIP.
5. Open **Site Intelligence → Live Intelligence**.
6. Confirm placement is **Below Astra breadcrumb** and parchment navigation is enabled.
7. Verify hover, keyboard focus, and the pause button stop the ticker.
8. Confirm the feed shows public event/weather information and no demonstration records.
9. Add `[sc_live_intelligence]` to a test page to verify shortcode placement.
