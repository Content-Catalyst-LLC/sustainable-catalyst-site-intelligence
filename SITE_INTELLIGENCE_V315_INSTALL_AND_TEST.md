# Site Intelligence v3.1.5 — Install and Test

1. Run the macOS installer and deploy the latest `main` commit on Render.
2. Upload the generated v3.1.5 WordPress plugin ZIP.
3. Open **Settings → SC Site Intelligence → Live Intelligence**.
4. Leave **Mobile presentation** set to **Signal rotator with previous / next controls**.
5. Test previous, next, swipe, pause, auto-advance, screen rotation, reduced motion, and the Hide on mobile option.
6. Confirm desktop remains a continuous electronic ticker.
