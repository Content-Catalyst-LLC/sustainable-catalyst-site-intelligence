# Report Generator and Export Intelligence

v0.7.0 adds internal report generation for Sustainable Catalyst Site Intelligence.

## Reports

- Site Intelligence Report
- Search Intelligence Report
- Content Strategy Report
- External Sources Report
- Climate + Energy Snapshot Report
- Registry and Indexing Coverage Report
- Export Bundle

## Formats

- JSON for programmatic use
- Markdown for planning notes and GitHub/editorial drafts
- CSV for spreadsheet review

## Privacy

Reports can contain internal planning details, raw recommendations, strategy queues, and operational diagnostics. They should remain private unless manually reviewed.
