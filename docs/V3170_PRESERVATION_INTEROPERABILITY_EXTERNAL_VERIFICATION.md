# v3.17.0 — Preservation Interoperability, External Verification, and Institutional Exchange

Site Intelligence prepares standards-aligned exchange packages from approved custody records. Profiles provide BagIt-style manifests, OAIS SIP mapping, PREMIS-like event metadata, and RO-Crate metadata mapping without claiming external certification. Independent verification receipts are human-reported and checksum-bound. No remote deposit, network verification, destination write, credential storage, or archive mutation occurs.
