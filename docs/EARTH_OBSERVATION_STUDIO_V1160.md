# Site Intelligence v1.16.0 — Earth Observation Studio

Earth Observation Studio is the first flagship visual workspace in Site Intelligence.

## Public experience

- before-and-after satellite imagery comparison
- draggable swipe divider
- synchronized maps
- layer opacity
- date controls
- recent timeline playback
- source metadata
- shareable view state
- PNG snapshot attempt
- print and JSON export

## Layers

- corrected-reflectance true color
- land-surface temperature
- fires and thermal anomalies
- vegetation index
- global precipitation
- snow cover
- nighttime lights
- atmospheric aerosols

## Interpretation boundary

Imagery products differ in latency, resolution, compositing, cloud sensitivity, and source availability. Apparent visual change does not establish cause. These layers support public orientation and research, not operational alerts or safety-critical decisions.
