# Site Intelligence v2.6.0 — Trade, Energy, and Resource Security Observatory

Public route: `/app/?view=resources`

This release organizes free official trade, energy, agriculture, food, water, materials, supply-chain, and transition records through the Platform Core public bridge. Units, periods, directions, counterpart geographies, classifications, frequency, license, attribution, and source links remain visible. Coverage is never converted into a proprietary risk score.
