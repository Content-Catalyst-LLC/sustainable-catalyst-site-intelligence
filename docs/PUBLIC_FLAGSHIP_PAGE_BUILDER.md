# Public Flagship Dashboard Page Builder

Site Intelligence v0.10.0 adds a public-safe page-builder layer for assembling the flagship Sustainable Catalyst Site Intelligence dashboard.

## Primary shortcode

```text
[sc_site_intelligence_public_flagship]
```

This renders a polished public dashboard page using only reviewed public modules.

## Builder / review shortcodes

```text
[sc_site_intelligence_public_page_builder]
[sc_public_dashboard_shortcode_bundle]
```

Keep these on internal or review pages unless intentionally exposing placement guidance.

## Public-safe modules

```text
[sc_site_intelligence_public_landing]
[sc_public_site_intelligence]
[sc_public_knowledge_overview]
[sc_public_climate_energy_summary]
[sc_public_methodology]
```

Do not use private analytics, admin diagnostics, reports, Search Console panels, or internal AI briefs on public pages unless manually reviewed.
