# Site Intelligence v1.12.1 — Dashboard Launch Polish

This release turns the configuration-driven Dashboard Studio into a launch-ready public experience.

## Public experience
- dashboard navigation and plain-language introduction
- responsive cards, filters, tables, and action controls
- loading skeletons and accessible live-region status text
- clear empty, stale, last-known-good, and unavailable states
- keyboard-visible focus and accessible table fallbacks
- stable share and export states
- source, freshness, methodology, and uncertainty notes

## Production QA
The backend reports launch readiness, but final browser, cache, and performance checks must be completed after Render and WordPress deployment.
