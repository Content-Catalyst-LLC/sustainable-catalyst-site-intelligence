# Site Intelligence v1.17.0 — Unified Live Event Intelligence

## Connected sources

- USGS Earthquake Hazards Program
- NASA EONET
- ReliefWeb humanitarian reports

## Public experience

- synchronized map and event list
- date-range filters
- category filters
- source filters
- optional ISO country filter
- event-detail drawer
- timeline playback
- category and source summaries
- source-aware interpretation boundaries

## Data boundaries

The event explorer reflects records returned by connected public sources. It does not represent complete incidence, severity, causality, or emergency instructions. Fallback demonstration records are explicitly labeled.
