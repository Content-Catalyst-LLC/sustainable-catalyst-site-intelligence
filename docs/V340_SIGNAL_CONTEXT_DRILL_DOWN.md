# v3.4.0 Signal Context and Drill-Down

Each Live Intelligence signal receives stable public context and evidence routes. Context is assembled from the current public-safe feed and preserves source identity, time, data state, ranking explanation, cluster lineage, and limitations.

## Context contract

A context packet includes:

- the selected signal;
- classification and development state;
- transparent selection rank, score, reasons, and components;
- primary and represented source records;
- observed, updated, and selection timestamps;
- source coordinates and an OpenStreetMap handoff when available;
- related Site Intelligence workspace routes;
- research suggestions based on visible term overlap only;
- an embedded evidence record and canonical SHA-256 digest;
- Site Intelligence and Decision Studio handoffs;
- responsible-use boundaries.

## Non-claims

The context layer does not claim independent verification, source accuracy certification, causal inference, improved geographic precision, emergency authority, or automatic publication readiness.
