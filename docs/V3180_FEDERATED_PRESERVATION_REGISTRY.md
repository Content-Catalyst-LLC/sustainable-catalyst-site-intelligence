# Site Intelligence v3.22.0 — Federated Preservation Registry, Trust Profiles, and Cross-Institution Verification

The federated preservation registry converts approved institutional exchange participation into public-safe organizational profiles and checksum attestations. It is an append-only governance layer and does not certify, rank, accredit, endorse, contact, or remotely control participating institutions.

Consensus is calculated from the latest public attestation from each unique approved institution. A verified result requires the approved exchange package checksum. Repeated attestations from one institution cannot increase the unique-institution count.
