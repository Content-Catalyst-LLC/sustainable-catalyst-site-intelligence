# Site Intelligence v1.12.6 — Public Interface Redesign and Renderer Simplification

This release replaces generic connector diagnostics in the primary public views with dedicated templates for Country Intelligence, Public Intelligence Dashboards, Cross-Domain Comparison, and the Dashboard Directory.

The redesign was driven by production PDFs showing duplicated embedded modules, narrow-column overflow, repeated ready/review labels, and technical source states occupying the main visual hierarchy.

No unavailable statistic is fabricated or silently imputed.
