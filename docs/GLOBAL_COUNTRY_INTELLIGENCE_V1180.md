# Site Intelligence v1.18.0 — Global Country Intelligence

## Scope

Global Country Intelligence uses the World Bank country catalog for searchable global ISO coverage and the existing live indicator pipeline for country metrics and trends.

## Public experience

- country search by name or ISO code
- regional filtering
- country identity and metadata
- geographic fly-to
- live indicator cards
- multi-year trends
- country-linked public events
- direct links into Earth Observation and Event Intelligence
- shareable country views

## Boundaries

World Bank reporting years vary by indicator. Missing values are not silently imputed. Country views organize evidence and do not create a national score, ranking, causal judgment, legal conclusion, or professional recommendation.
