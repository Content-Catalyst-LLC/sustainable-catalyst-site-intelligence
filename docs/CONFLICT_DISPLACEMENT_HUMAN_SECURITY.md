# Conflict, Displacement, and Human Security — v1.11.0

This release establishes public-safe contracts for conflict events, violence against civilians, infrastructure disruption, forced displacement, humanitarian access, and modeled risk.

## Sources

- UCDP
- UNHCR Refugee Data Finder

## Governance

Observed events, allegations, verified source findings, official statistics, estimates, and modeled forecasts remain distinct. The platform does not infer guilt, combatant status, refugee status, eligibility, or legal responsibility. Sensitive locations must be generalized where disclosure could increase risk.
