# Site Intelligence v1.14.1

## Immersive Visual System and Application Polish

This release establishes the production visual language for the standalone Site Intelligence application.

### Visual foundation

- deep charcoal map-first shell
- Sustainable Catalyst red as the primary selection accent
- atmospheric blue and cyan for Earth-observation context
- translucent evidence panels
- cinematic map vignette and contextual caption
- restrained depth, shadows, and motion

### Interaction polish

- animated event markers
- skeleton loading states
- route transitions
- metric reveal motion
- hover and focus refinements
- fullscreen map
- shareable country and view state
- reduced-motion support

### Product boundary

The release intentionally adds no fabricated country metrics and does not expand source claims. It improves the visual and interaction layer around the existing validated public endpoints.
