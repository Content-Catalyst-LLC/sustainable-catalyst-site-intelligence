# Sustainable Development Data Connectors and Public Source Registry

Version 1.6.0 establishes a common source and observation contract for planetary boundaries, SDGs, poverty, education, food systems, water, climate, and human development.

## Public backend endpoints

- `/public/sustainable-development/sources`
- `/public/sustainable-development/families`
- `/public/sustainable-development/planetary-boundaries`
- `/public/sustainable-development/health`
- `/public/sustainable-development/methodology`

## WordPress shortcodes

- `[sc_public_sustainable_development_sources]`
- `[sc_public_sustainable_development_families]`
- `[sc_public_planetary_boundaries_registry]`
- `[sc_public_sustainable_development_source_health]`
- `[sc_public_sustainable_development_methodology]`

## Boundary

Current observations, published scientific thresholds, and Sustainable Catalyst derived assessments are separate record types. The platform must not present a derived comparison as an official planetary-boundary assessment.
