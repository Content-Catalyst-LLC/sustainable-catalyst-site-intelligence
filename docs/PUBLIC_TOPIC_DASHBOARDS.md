## Public/private boundary

Topic dashboards are public-safe orientation surfaces. They should not expose raw analytics, Search Console query details, credentials, API payloads, force-refresh diagnostics, internal publishing queues, or unreleased AI-assisted briefs.

Use these topic dashboards as reviewed public pages or embedded modules alongside methodology and boundary language.
