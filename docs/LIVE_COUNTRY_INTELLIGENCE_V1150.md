# Site Intelligence v1.15.0 — Live Country Intelligence

v1.15.0 turns Country Intelligence from a source registry into a working public evidence view.

## Live evidence

The backend queries World Bank Open Data for:

- population
- life expectancy
- GDP per capita
- access to electricity
- basic drinking water
- secondary enrollment
- CO2 emissions per capita
- Gini index

Every record retains its reporting year, unit, source, indicator identifier, and data state.

## Country application view

The standalone application now displays:

- six prominent country indicators
- live/reference state
- multi-year trend selector
- source and methodology notes
- satellite and event context already provided by the map workspace

## Boundaries

Reporting years differ. Missing values are not imputed. Country indicators do not create a national score or establish causality, responsibility, eligibility, or professional advice.
