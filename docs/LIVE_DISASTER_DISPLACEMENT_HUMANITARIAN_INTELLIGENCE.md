# Live Disaster, Displacement, and Humanitarian Intelligence

Version 1.8.0 registers GDACS, ReliefWeb, USGS real-time earthquake feeds, NASA EONET, and UNHCR Refugee Statistics. The public layer separates physical hazards, preliminary alerts, curated humanitarian reporting, and periodic official displacement statistics. It is informational and is not an emergency-warning, immigration-status, humanitarian-eligibility, or legal-advice service.
