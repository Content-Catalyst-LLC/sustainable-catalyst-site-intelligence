# Sustainable Catalyst Site Intelligence v2.12.0

## Offline, Mobile, Accessibility, and Performance

Adds an installable application manifest, browser service worker, offline fallback page, bounded browser-local cache policy, low-bandwidth controls, mobile-safe delivery, accessibility contracts, public performance budgets, and a dedicated experience workspace. Cached records remain explicitly non-authoritative, external map and imagery services may remain network-dependent, and automated checks do not claim formal accessibility certification.
