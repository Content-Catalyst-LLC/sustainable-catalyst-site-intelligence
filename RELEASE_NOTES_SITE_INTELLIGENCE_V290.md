# Sustainable Catalyst Site Intelligence v2.9.0

## Comparative Intelligence and Scenario Studio

Adds multi-geography indicator baskets, explicit compatibility review, regional peer context, transparent arithmetic scenarios, correlation review, and reproducible JSON packets. No paid API is required. The studio does not silently normalize incompatible values, create rankings, forecast outcomes, infer causation, or provide investment or policy recommendations.

Public route: `/app/?view=scenarios`

WordPress shortcode: `[sc_comparative_intelligence_scenario_studio height="1550"]`
