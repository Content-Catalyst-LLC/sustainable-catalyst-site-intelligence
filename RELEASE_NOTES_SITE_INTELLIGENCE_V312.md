# Site Intelligence v3.1.2 — Navigation Surface Harmony and Signal Expansion

## Summary

This release broadens Live Intelligence while returning navigation and breadcrumb colors to the active Astra/theme configuration. The plugin does not paint, recolor, or normalize those site surfaces.

## Signal expansion

- Default ticker capacity: 16 signals.
- Configurable maximum: 24 signals.
- Maximum selected signals from one source: 3.
- Category rotation across Earth systems, human systems, research, economy/resources, and platform context.
- Additional verified signal families: strongest recent earthquake, natural-event category summary, humanitarian geographic coverage, recent NASA POWER observations, recent open-access research metadata, and periodic World Bank indicators.
- Periodic indicators display their reporting year and are not represented as real-time measurements.

## Interface behavior retained

- Automatic placement below Astra's configured breadcrumb.
- Below-header fallback and shortcode-only placement.
- `[sc_live_intelligence]` shortcode.
- Manual pause plus hover and keyboard-focus pause.
- Cached last-known-good feed continuity.
- Black electronic-board presentation with green text.

## Theme boundary

v3.1.2 removes the v3.1.1 parchment navigation option and its CSS selectors. Existing Astra/theme values determine the secondary navigation and breadcrumb appearance.
