# Site Intelligence v3.1.5 — Mobile Navigation and Motion Repair

## Summary

v3.1.5 replaces the unreliable mobile marquee default with a one-signal-at-a-time electronic rotator. It preserves the desktop ticker and all source, placement, taxonomy, cache, and shortcode behavior.

## Mobile repair

- Previous and next buttons are always visible in rotator mode.
- A current-position counter reports the active signal.
- Signals auto-advance every seven seconds by default.
- Horizontal swipe gestures move backward and forward.
- Pause stops both desktop marquee motion and mobile auto-advance.
- Reduced-motion preferences disable auto-advance while retaining manual navigation.
- Touch devices no longer inherit the desktop hover-pause behavior.
- Administrators can choose Rotator, Scrolling ticker, or Hidden on mobile.

## Boundaries

No navigation or breadcrumb colors are changed. No feed-selection, placement, source-attribution, or data-contract behavior is removed.
