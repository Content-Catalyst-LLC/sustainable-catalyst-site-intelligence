# Site Intelligence v3.3.0 — Install and Test

1. Put `install_and_push_site_intelligence_v3_3_0.sh` and `sustainable-catalyst-site-intelligence-v3.3.0-repo.zip` in `~/Downloads`.
2. Run the installer from Terminal.
3. Deploy the latest `main` commit on Render.
4. Upload `sustainable-catalyst-site-intelligence-v3.3.0-wordpress.zip` in WordPress.
5. Confirm backend/plugin version parity.
6. Open `/public/live-intelligence/ranking-policy` and verify the policy is available.
7. Confirm the ticker shows represented-source counts only for multi-source clusters.
8. Hover a signal and confirm its tooltip includes development state and selection reasons.
9. Verify desktop pause, mobile previous/next, swipe, and feed-selection controls remain functional.
