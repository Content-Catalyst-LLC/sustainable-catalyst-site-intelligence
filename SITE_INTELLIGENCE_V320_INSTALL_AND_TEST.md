# Site Intelligence v3.3.0 install and test

1. Run the macOS installer beside the v3.3.0 repository ZIP.
2. Deploy the pushed `main` commit on Render.
3. Upload the generated v3.3.0 WordPress ZIP.
4. Confirm WordPress and backend both report 3.3.0.
5. Open **Settings → SC Site Intelligence → Live Intelligence**.
6. Confirm the ticker still displays and the mobile rotator still advances.
7. Review **Signal source operations**.
8. Run **Check config** on each enabled source.
9. Run live tests selectively after confirming the backend API token and external-live settings.
10. Verify no navigation or breadcrumb color changes occurred.
