# Site Intelligence v3.19.0 — Install and Test

Run the macOS installer from the release bundle. The default gate runs the v3.12.0–v3.19.0 preservation workflow tests plus the v3.19.0 release contract. Set `SC_SI_FULL_TESTS=1` to run the complete backend suite.

After deployment, use `[sc_live_intelligence_preservation_exchange]` for the read-only public surface.
