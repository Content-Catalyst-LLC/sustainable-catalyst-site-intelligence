# Changelog

## v0.6.0 — Advanced External Data Connectors

- Added NOAA / NWS weather and climate-context connector.
- Added EIA energy connector with optional API key support.
- Added EPA AQS air-quality connector with optional credentials.
- Added U.S. Census context connector.
- Added USGS land-cover context connector.
- Added GBIF biodiversity occurrence connector.
- Added advanced external connector health endpoint.
- Added Environmental Monitoring dashboard.
- Added Urban Resilience dashboard.
- Added Biodiversity / Land Use dashboard.
- Added Energy Systems dashboard.
- Added WordPress shortcodes for the new dashboards.
- Preserved stable fallback behavior so dashboards do not fail when optional external credentials are missing.
