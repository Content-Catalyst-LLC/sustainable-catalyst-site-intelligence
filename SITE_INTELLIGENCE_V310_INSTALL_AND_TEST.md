# Site Intelligence v3.1.0 — Install and Test

1. Run the included macOS installer from Downloads.
2. Deploy the updated `main` branch on Render.
3. Upload the generated WordPress ZIP.
4. Open **Settings → SC Site Intelligence**.
5. Confirm **Live Intelligence** and **Top electronic ticker** are enabled.
6. Keep **Homepage only** as the initial scope.
7. Verify `/public/live-intelligence` and the homepage board.

The shortcode `[sc_live_intelligence]` remains usable when automatic top placement is off.
