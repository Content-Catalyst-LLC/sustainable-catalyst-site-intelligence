# Site Intelligence v3.1.2 — Install and Test

1. Put the v3.1.2 repository ZIP and installer in `~/Downloads`.
2. Run `chmod +x install_and_push_site_intelligence_v3_1_2.sh`.
3. Run `./install_and_push_site_intelligence_v3_1_2.sh`.
4. Deploy the latest `main` commit on Render.
5. Upload the v3.1.2 WordPress plugin ZIP.
6. Clear page/CDN caches and reload the homepage.
7. Confirm the secondary navigation and breadcrumb use the site's existing Astra/theme values.
8. Confirm Live Intelligence shows a broader, balanced feed and still pauses on hover, focus, and button activation.
9. Confirm `[sc_live_intelligence limit="16"]` works on a test page.
