# Site Intelligence v3.14.0

## Corrections, Retractions, and Public Change History

This release converts approved public correction packages into separately reviewed, append-only public change notices.

### Added

- Correction, clarification, replacement, retraction, and rollback notices.
- Public release lineage showing incoming and outgoing changes.
- Explicit original-release retention for every notice.
- Separate approval for public change notices.
- Retraction acknowledgement that the original record remains retained.
- JSON and Markdown change packages.
- Manual Publications, Knowledge Library, WordPress-package, and download handoffs.
- Public policy, status, history, and release-lineage endpoints.
- Read-only WordPress change-history shortcode.

### Boundaries

- No evidence rewrite.
- No destination write.
- No destination deletion.
- No silent replacement.
- No automatic republication.
- No credentials or recipient identities stored.

The original release remains retained and addressable in the append-only history.
