# Site Intelligence v3.1.5 installation and test guide

1. Place the repository ZIP and installer in `~/Downloads`.
2. Run the installer.
3. Deploy the latest `main` commit on Render.
4. Upload the generated v3.1.5 WordPress plugin ZIP.
5. Open **Settings → SC Site Intelligence**.
6. Review **Ticker speed**, **Custom cycle**, **Story spacing**, **Maximum signal text**, **Source labels**, and **Category taxonomy**.
7. Confirm the homepage ticker pauses on hover, keyboard focus, and the pause button.
8. Confirm the automatic ticker and any `[sc_live_intelligence]` shortcodes still respect saved feed and placement settings.

Recommended starting mode: **Balanced — 30s desktop / 36s mobile**.
